# 🚀 Digital Traffic Hub

Site Next.js pour la vente de produits digitaux et marketing via reels.

## 📌 Fonctionnalités
- Page d’accueil avec vidéo YouTube intégrée
- Section produits digitaux
- Blog marketing
- Page contact

# Installe les dépendances
npm install

npm run dev
